# fatfat
[<img src="https://raw.githubusercontent.com/gguf-org/pig/master/pig.gif" width="128" height="128">](https://github.com/gguf-org/fat)

view, inspect and edit GGUF model file metadata and tensor info directly in your browse even without internet (offline)

## choose your browser

copy the link below to your chrome browser
```
chrome://extensions/
```

copy the link below to your edge browser
```
edge://extensions/
```

copy the link below to your brave browser
```
brave://extensions/
```

## sideload to your browser as an extension
- turn on `Developer mode`
- click `Load unpacked` and select this repo clone

## or an easy way, search for it from your browser extension marketplace/store

for examples, chrome: [gguf](https://chromewebstore.google.com/detail/gguf/holnhkpcfodagjnlkkoiaamidblmmlba), [fat](https://chromewebstore.google.com/detail/fat/hbndjojdmfeljamfmoijeaeabiflchbh); firefox: [gguf](https://addons.mozilla.org/en-US/firefox/addon/gguf/), [fat](https://addons.mozilla.org/en-US/firefox/addon/fat/); edge: [fat](https://microsoftedge.microsoft.com/addons/detail/fat/pfheocmngehhcomcdoimpbhphjodbinl), and etc.
